# 📊 Data Analytics Projects
  
This repository contains three end-to-end data analysis projects using Python. These projects focus on data cleaning, exploratory data analysis (EDA), visualization, and deriving meaningful business insights. 
 
---

## 🏥 1. Healthcare Data Analysis
 
**Objective:**
Analyze patient data to identify health patterns and high-risk patients.

**Key Work:**

* Data cleaning (missing values, duplicates)
* Feature engineering (Age Group, Risk Level)
* EDA using Pandas
* Visualization using Matplotlib & Seaborn

**Insights:**

* High BMI patients are at greater health risk
* Lifestyle factors like smoking and alcohol impact health
* Certain cities have higher patient concentration

---

## 🏨 2. Hotel Booking Analysis

**Objective:**
Analyze booking data to understand customer behavior, cancellations, and revenue (ADR).

**Key Work:**

* Data cleaning and preprocessing
* Handling missing values (children, agent, company)
* Feature engineering (total guests)
* Comparative analysis of City vs Resort hotels

**Insights:**

* High cancellation rate observed
* City hotels have more bookings
* Bookings show seasonal trends
* Lead time impacts cancellation behavior

---

## 🚲 3. Bike Demand Analysis

**Objective:**
Analyze bike rental demand based on time, weather, and seasonal factors.

**Key Work:**

* Data preprocessing and date conversion
* Feature engineering (Year, Month, Day)
* EDA and visualization
* Correlation analysis

**Insights:**

* Demand peaks during working hours
* Weather conditions significantly affect demand
* Temperature positively influences bike rentals
